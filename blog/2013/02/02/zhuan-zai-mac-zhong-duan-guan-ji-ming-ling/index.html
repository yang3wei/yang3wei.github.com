(转载)Mac 终端关机命令

Feb 2nd, 2013 | Comments

原文链接：
http://pulading1988.diandian.com/post/2012-04-22/17392701
1.立即关机：
sudo halt 或 sudo shutdown -h now

2.10 分钟后关机：
sudo shutdown -h +10

3.晚上 8 点关机：
sudo shutdown -h 20:00

4.立即重启：
sudo reboot 或者 sudo shutdown -r now
